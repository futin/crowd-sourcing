#  Crowd Sourcing

> a backend implementation of a crowd-sourcing app

## Technology stack

    - NodeJs
    - GraphQL
    - MongoDB
    - Passport Google Auth
    - Docker (upcoming)

## Install

Please run the "install" command first:

```sh
$ npm install
```

This will install all of the required dependencies, as stated in `package.json`file.
